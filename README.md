# Черричка: Архитектура революции

Тактический мини‑прототип на `HTML5 Canvas`: вы **не управляете героем напрямую** — Чиполлино сам идёт к выходу, а вы меняете архитектуру уровня инструментами «садовода‑архитектора».

## Краткий обзор

**Черричка: Архитектура революции** — это тактическая головоломка‑платформер в неоновой стилистике, где победа достигается не силой, а **перепроектированием маршрута**: строите временные платформы, срезаете свои же конструкции и управляете водными потоками через клапаны.

Ваша цель на уровне: **провести Чиполлино к выходу**, не дав его настичь Лимонишкам и не уронив на шипы.

## Ключевые особенности

| Особенность | Описание |
|---|---|
| 3 уровня | Последовательно усложняющиеся карты с разными препятствиями |
| 3 инструмента архитектуры | **Взрастить платформу**, **Подрезать**, **Клапан полива** |
| «Гнилой Устав» (распад построек) | Построенные платформы существуют ограниченное время (TTL) |
| Враги‑патрули | Лимонишки двигаются по платформам, разворачиваются на краях и могут срезать ваши растения |
| Вода как оружие | Потоки воды от клапана мгновенно «смывают» врагов на линии потока |
| Самодостаточный проект | Вся игра — один файл `index.html`, без зависимостей |

## Геймплей

### Основная механика

- **Герой** (Чиполлино) двигается к выходу автоматически.
- Вы выбираете инструмент и применяете его **ЛКМ** по клетке/объекту.
- Задача — создать безопасный путь и/или устранить врагов средой.

### Инструменты

| Инструмент | Как использовать | Эффект |
|---|---|---|
| 1 — Взрастить платформу | ЛКМ по пустой клетке рядом с героем | Создаёт временную платформу‑«подсолнух» |
| 2 — Подрезать | ЛКМ по вашей платформе | Удаляет платформу (можно «сбросить» врага) |
| 3 — Клапан полива | ЛКМ по клапану | Переключает направление воды и создаёт поток (смывает врагов на линии) |

## Управление

| Клавиша/мышь | Действие |
|---|---|
| `1` `2` `3` | Выбор инструмента |
| `ЛКМ` | Применить выбранный инструмент |
| `R` | Перезапуск уровня |
| `N` | Следующий уровень |
| `Space` | Пауза |

## Установка и запуск

### Требования

- Любой современный браузер (Chrome/Edge/Firefox)

### Запуск

- Откройте `index.html` двойным кликом

Альтернатива (локальный сервер):

```bash
python -m http.server 8000
```

Затем откройте:

- `http://localhost:8000`

## Прогрессия

Проект включает **3 уровня**:

| Уровень | Фокус | Особенности |
|---|---|---|
| 1 | Базовая стройка | Пробуйте строить мосты и обходы |
| 2 | Вода/клапаны | Используйте клапан, чтобы смывать врагов |
| 3 | Вертикальность | Стройте «башни» из временных платформ, обходите патрули |

## Игровые системы (внутри)

- **Сетка тайлов** `40x22`, тайл `32px`.
- **Коллизии AABB** для перемещения персонажей.
- **Система размещаемых объектов** (`placed`) с временем жизни (TTL).
- **Вражеский патруль** с разворотом у препятствий и попыткой срезать размещённые платформы.
- **Водные лучи** от клапана (временные «beams»), уничтожающие врагов на линии.
